# MedAgent-Report-Generation-Query-Alloydb-using-MCP-Toolbox-and-Gemini-CLI-
This Application queries Alloydb and generates answers to the user query. It uses MCP Toolbox for databases for getting connected to the data source and generates a comphrehensive report in pdf format suggesting further treatment plan. Gemini CLI is used as a client to intereact with MCP server to get queries answered.
